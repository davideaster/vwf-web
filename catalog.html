adl/sandbox

Launch New Session

ADL's sandbox application allows the user to create and edit their own environment. Be sure to check out some of the demo environments - you can reach them from the "Visit an Existing World" button.


Kimray Valve

Launch New Session

KimRay Valve is a live 3-D Immersive Maintenance Demonstration built on top of Virtual World Framework to demonstrate the capabilities of Remote Subject Matter Experts interacting with a student using a shared real time lesson workspace utilizing webRTC to deliver real time video and voice over IP without any browser plugins. This demo requires the latest Firefox or Chrome browsers.

agi/cesium

Launch New Session

Cesium is a web-based globe and map engine for visualizing dynamic data. This demo application allows a user to have synchronized camera views across multiple clients. Pressing the "R" key will reset the synchronized views across all instances.

radio

Launch New Session

The radio application allows a user to click its buttons and knobs and the radio will update its screen accordingly. For example, clicking the top right most knob will cause the screen to turn on. Additionally, clicking the middle knob will cause the channels to change or clicking the red button will scroll through the backlight, beep function, sql, and channel functions. The side bottom buttons can update the backlight, beep, and sql functions. NOTE This application works best in Firefox.

Launch New Session

The tile puzzle application allows a user to shuffle the board or solve the board automatically. It also allows the user to play by clicking on a tile and the tile will move to the open space. The goal of the game is to have the numbers show in sequential order from left to right, top to bottom.
